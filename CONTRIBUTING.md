# Contribution Guidelines

Thank you for your interest in contributing to **Awesome AI for Education**!

## How to contribute

1.  **Fork** this repository.
2.  **Clone** your fork.
3.  **Add** your resource (Paper, Tool, Dataset) to the appropriate section in `README.md`.
    - Format: `- [Name](Link) - Description.`
    - Keep descriptions concise.
4.  **Commit** and **Push**.
5.  **Submit a Pull Request**.

## Rules
- Please ensure the link is active.
- Papers should link to arXiv or the official conference proceeding.
- Code repositories should link to GitHub/GitLab.
